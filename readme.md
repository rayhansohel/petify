# Petify - Pet Adoption Platform

**Petify** is a user-friendly platform for browsing and adopting pets. It fetches pet-related data through APIs, allowing users to explore pets by category, view detailed information, and adopt their new best friends!

## Key Features
1. **Responsive Navbar**: Adapts to different screen sizes, with a mobile-friendly menu.
2. **Dynamic Pet Listings**: Display pets based on categories fetched via API.
3. **Like & Adopt**: Users can like pets and adopt them with a countdown feature.
4. **Sort by Price**: Sort pets by price in descending order for better browsing.
5. **Pet Details Modal**: Displays detailed information when "Details" is clicked.

## APIs Used
- **Fetch All Pets**: `https://openapi.programming-hero.com/api/peddy/pets`
- **Fetch Pet Details by ID**: `https://openapi.programming-hero.com/api/peddy/pet/pet-id`
- **Fetch All Pet Categories**: `https://openapi.programming-hero.com/api/peddy/categories`
- **Fetch Pets by Category**: `https://openapi.programming-hero.com/api/peddy/category/categoryName`
